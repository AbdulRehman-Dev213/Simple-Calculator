# Simple Javascript Calculator

A simple calculator built using **HTML** and **JavaScript** to practice the fundamentals of JavaScript programming.

This is my **first JavaScript mini-project**, created to strengthen my understanding of core programming concepts such as variables, objects, conditionals, operators, and user input handling.

---

# 🚀 Features

* ➕ Addition
* ➖ Subtraction
* ✖️ Multiplication
* ➗ Division
* ⚠️ Division-by-zero error handling
* 🔢 Exponent (`**`) operation
* 💬 User input using the `prompt()` function
* 📄 Displays the result directly in the browser

---

# 📚 JavaScript Concepts Demonstrated

This project helped reinforce several fundamental JavaScript concepts, including:

* Variables
* Objects
* `if...else if...else` conditional statements
* Arithmetic operators (`+`, `-`, `*`, `/`, `**`)
* User input with `prompt()`
* Type conversion using `Number()`
* Basic error handling for invalid mathematical operations

# ▶️ How to Use

1. Clone  this repository.

2. Open `index.html` in your web browser.

3. Enter:

   * The first number
   * The second number
   * The desired operation

Supported operations:

```text
+
-
*
/
**
```

4. The calculator will display the answer directly in the browser.

---

# 🛠 Example

```text
Enter Number 1: 20

Enter Number 2: 5

Enter Operation:
/

Output:
Answer is: 4
```

If division by zero is attempted:

```text
Error: cannot divide by 0
```
